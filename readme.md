### Installation

Add repositories to `composer.json`:
```json
{
  "repositories": [
    {
      "type": "vcs",
      "url": "https://github.com/stylemix/laravel-base.git"
    }
  ]
}
```

#### For Laravel 6.x
Require packages:
```bash
composer require stylemix/laravel-base
```

#### For Laravel 5.8.x
```bash
composer require stylemix/laravel-base:~0.5.1
```
